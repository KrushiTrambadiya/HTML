Assignment Questions and Answers

  1. What are nested tags in HTML?

    Nested tags are HTML tags that are placed inside one another to create a structured hierarchy and allow for more complex formatting.

  2. Can all HTML tags be nested?

    No, not all HTML tags can be nested. Some tags are block-level and cannot contain inline elements.

  3. What is the purpose of the <strong> tag?

    The <strong> tag is used to indicate that the text is of strong importance, usually rendering it as bold.

  4. How does the <em> tag differ from the <i> tag?

    The <em> tag is used for emphasizing text and is typically rendered as italic, while <i> is used purely for italicizing text without any semantic meaning.

  5. What effect does the <u> tag have?

    The <u> tag underlines the text it wraps.

  6. What is the difference between <del> and <ins> tags?

    The <del> tag indicates deleted text, while the <ins> tag indicates inserted text.

  7. What is the <mark> tag used for?

    The <mark> tag is used to highlight text, making it stand out.

  8. How can you create superscript text?

    Superscript text can be created using the <sup> tag.

  9. What is the purpose of the <small> tag?

    The <small> tag is used to display text in a smaller font size.

  10. Why are text formatting tags important?

    Text formatting tags enhance the readability and structure of the content, allowing users to understand the significance of different parts of the text.
