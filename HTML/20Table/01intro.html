Practical Use

The <table> tag is useful for:

  • Displaying structured data in rows and columns, such as financial reports, schedules, and grading systems.
  • Organizing data that needs to be compared or categorized, making it easier for users to read and understand.
  • Presenting information in a visually appealing manner, which can enhance user experience.

HTML Tags

Descriptions
<table> Defines the structure for organizing data in rows and columns within a web page.
<tr> Represents a row within an HTML table, containing individual cells.
<th> Shows a table header cell that typically holds titles or headings.
<td> Represents a standard data cell, holding content or data.
<caption> Provides a title or description for the entire table.
<thead> Defines the header section of a table, often containing column labels.
<tbody> Represents the main content area of a table, separating it from the header or footer.
<tfoot> Specifies the footer section of a table, typically holding summaries or totals.
<col> Defines attributes for table columns that can be applied to multiple columns at once.
<colgroup> Groups together a set of columns in a table to which you can apply formatting or properties collectively.

Assignment Questions and Answers

  1. What is the purpose of the <table> tag in HTML?

    The <table> tag is used to create a table for displaying data in rows and columns.

  2. What are the key elements used within a table?

    The key elements are <tr> (table row), <th> (table header), and <td> (table data cell).

  3. How do you define a table header?

    A table header is defined using the <th> tag, which typically appears within a <tr> element.

  4. What is the difference between <th> and <td>?

    The <th> tag is used for header cells (bold and centered by default), while the <td> tag is used for standard data cells.

  5. Can a table have a caption?

    Yes, you can use the <caption> tag to provide a title or description for the table.

  6. How do you style a table using CSS?

    You can apply styles to the <table>, <th>, and <td> elements to change appearance, such as borders, padding, and background colors.

  7. Is it possible to merge cells in a table?

    Yes, you can use the colspan attribute in the <td> tag to merge columns and rowspan to merge rows.

  8. What is the border-collapse property in CSS?

    The border-collapse property controls the behavior of borders in tables, allowing you to combine borders of adjacent cells into a single border.

  9. How do you ensure accessibility for tables?

    To ensure accessibility, use the <caption> element for a table title and consider using the scope attribute in header cells to specify the relationship to data cells.

  10. Can you include images or links within table cells?

    Yes, you can include any HTML elements, such as images or links, within table cells to enhance the content.
